Human Language

Human Language is a readable programming language with plain-English syntax, VS Code support, and a published Python package.

Get started

  • Install from PyPI: pip install human-language
  • Run programs with: human program.hm
  • Use the VS Code extension for syntax highlighting and editor support

Why Human?

Plain English syntax
Readable language constructs that feel natural to write.
VS Code support
Syntax highlighting and language recognition for .hm files.
Python packaging
Published as human-language on PyPI.
Browser-friendly docs
Searchable documentation and examples.
